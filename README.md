# qq-farm-bot | QQ经典农场机器人  
# 帮你实现全自动种菜、收菜、偷菜等功能，支持QQ端与VX端，方便快捷的前端操作界面 

# 项目说明
- 本项目主要是基于 [Penty-d/qq-farm-bot-ui](https://github.com/Penty-d/qq-farm-bot-ui) 项目进行了安全性方面的升级优化，如你不需要考虑安全问题可直接部署使用原项目即可，本项目目的是尽力避免大家在云服务器等公网环境部署使用时出现安全风险问题，具体安全优化日志详见[Update_security.log](./Update_security.log)  

- 本项目会尽可能地及时同步原项目的其它功能更新，具体更新项详见[Update.log](./Update.log)，并在此基础上会尝试融入其它类似项目较优秀的功能机制

- 有任何问题或建议，欢迎提issue，沟通交流Q群：1090602010

# 功能特性

### 安全性提升(相较原项目)
- 前后端密码加密传输、存储（原项目为明文传输）
- 完善的登录token生命管理周期（原项目为永久token，即登录一次永久有效）
- 无初始默认管理密码机制（原项目为默认admin密码需手动修改）
- 其它安全问题正在完善提升中...

### 多账号管理
- 账号新增、编辑、删除、启动、停止
- 手动输入 Code
- 账号被踢下线自动删除
- 账号连续离线超时自动删除
- 账号离线推送通知（支持 Bark、自定义 Webhook 等）

### 自动化能力
- 农场：收获、种植、浇水、除草、除虫、铲除、土地升级
- 仓库：收获后自动出售果实
- 好友：自动偷菜 / 帮忙 / 捣乱
- 任务：自动检查并领取
- 好友黑名单：跳过指定好友
- 静默时段：指定时间段内不执行好友操作

### Web 面板
- 概览 / 农场 / 背包 / 好友 / 分析 / 账号 / 设置页面
- 实时日志，支持按账号、模块、事件、级别、关键词、时间范围筛选
- 深色 / 浅色主题切换

### 分析页
支持按以下维度排序作物：
- 经验效率 / 普通肥经验效率
- 净利润效率 / 普通肥净利润效率
- 等级要求

### 更多功能持续更新优化中...

# 部署使用方法
## 环境要求及部署启动流程
考虑到不同人的部署环境配置不同，因此强烈建议通过docker进行部署启动，如果非要通过源码启动，请参考原项目说明，此处不再赘述  

1、确保有Docker环境（如何安装部署docker请自行查阅其它教程）  
2、 在终端内进入本项目目录，执行以下命令
```bash
# 构建并后台启动
docker compose up -d --build
# 若输出如下则说明启动成功：
# [+] Running 1/2
# [+] Running 2/3-ui                  Built                       
# ..........
# ..........
#  ✔ qq-farm-bot-ui                  Built                                                     
#  ✔ Container qq-farm-bot-ui        Started                                                  
```
3、启动成功后通过浏览器访问  
- 本机：`http://localhost:3000`
- 其它机器：`http://<你部署设备的IP>:3000`

## 版本更新方法
1、进入项目目录并打开终端  
2、执行以下命令 
```bash
# 停止关闭并删除现有容器
docker compose down

# 拉取最新代码
git pull

# 重新构建并后台启动
docker compose up -d --build
```
3、更新完成，访问网页即可继续使用

## 查看后端日志方法
1、进入项目目录并打开终端  
2、执行以下命令
```bash
# 显示实时日志
docker compose logs -f
```
